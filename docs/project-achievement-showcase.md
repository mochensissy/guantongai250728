| **用户控制** | 系统决定 | 用户完全控制 | **极佳用户体验** |

### 智能数据分类系统

```typescript
// 创新的四级数据分类
export type DataImportance = 
  | 'critical'   // 收藏卡片、用户偏好 → 必须同步
  | 'important'  // 活跃会话、学习进度 → 可选同步  
  | 'optional'   // 近期会话 → 完整同步
  | 'temporary'  // 过期对话 → 仅本地，定期清理
```
