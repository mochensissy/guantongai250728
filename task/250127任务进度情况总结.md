---

## 📋 项目概述

**项目名称**: AI学习私教引导平台  
**项目定位**: 智能文档学习引导平台，提供个性化AI私教服务  
**技术栈**: Next.js + React + TypeScript + Tailwind CSS + localStorage  
**核心理念**: 让AI成为用户的专属学习导师，提供苏格拉底式对话教学
